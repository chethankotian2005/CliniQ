🔍 Common Issues & Solutions

Issue 1: "Camera not starting"

  • Solution: Refresh page and click "Allow" when browser asks for camera permission
  • Alternative: Use the "Manual Entry" option instead
  • Check: Ensure no other apps are using the camera

Issue 2: "QR code not detected"

  • Solution: Ensure QR code is well-lit and clearly visible
  • Try: Move camera closer/farther from QR code
  • Alternative: Use "Manual Entry" and copy-paste the QR data

Issue 3: "Booking already processed"

  • Reason: QR code was already scanned before
  • Solution: This is normal - each QR can only be used once
  • Test: Use the fresh test QR code provided above

Issue 4: "Date mismatch error"

  • Reason: QR code is for a different date
  • Solution: Only scan QR codes for today's bookings
  • Test: Use the fresh test QR code created for today
